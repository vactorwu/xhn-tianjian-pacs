安装指南：
*************************************************
系统要求：
Microsoft Windows 9x/Me
Microsoft Windows NT 4.0
Microsoft Windows 2000 (推荐)
Microsoft Windows XP
*************************************************
步骤一：
执行Setup.exe进行安装；
选择安装界面的语言：中文简体、中文繁体、英文（仅应用于安装界面）；
阅读软件使用许可协议；
填入必要的用户信息；
填入产品序列号（此版本可随意输入）；
在安装类型中选择“自定义安装”，如选择“典型”或“最小安装”，则系统自动进行完全安装；
安装完成。
*************************************************
步骤二：
如果您的计算机第一次使用本软件，请进入安装目标文件夹（默认为C:\Program files\北京天健\Medvision\）中的“Drive”目录，安装加密狗驱动程序；
完成工作后，必须重新启动计算机！
*************************************************
步骤三：
使用本软件之前，请先对照软件使用手册中《基本设置》一章对影像服务程序和报告服务程序进行必要的设置，如果您安装了影像样本，可以在Medserver中将默认的d:\Image影像存储目录改为安装目标文件夹（默认为C:\Program files\北京天健\Medvision\）中的“Image”目录，并重新构造在线数据库，这样就可以使用软件自带的影像样本了。
*************************************************
